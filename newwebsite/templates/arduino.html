Arduino

What is an arduino?

I am no expert, infact im a beginner but ill explain it how i understand it. its essentially a tiny computer that can connnect input devices and output devices in any way you can possibly think of. You can code into it to process the inputs and manipulate the output devices. Heres a great youtube video i saw about it by mark rober/crash labs.

My build:

The goal for my build is to essentailly create a smart home appliance which can detect motion and switch on the light to my room. It will hopefully not only switch it one based of motion but also if its required (i.e if the room is sufficiently lit without it) to save electricity and for enviromental sustainability.

Now there are a couple different components to my build that may be useful to know such as:

  • Arduino: the heart of this entire thing which i already explained
  • PIR sensor: is an infa red sensor which im using as a motion detector
  • Jumper wires: the wire used to connect everything
  • Breadboard: The 'hub' where i connect everything, bascially how i see it, is a way to connect wiress without having to fuse them together.
  • LDR: a light dependant resistor, but in my case how im going to tell how much light is present
  • Servo motor: the end effector, or output device im going to use to switch on the lights

And its messy again.
